☁️ Sinkronisasi Data ke Google Sheets

Simpan dan sinkronisasi data audit & evaluasi dosen Anda ke Google Sheets secara otomatis.

📌 Cara Setup (3 Step):
1. Buat Google Apps Script (lihat panduan di bawah)
2. Copy URL Web App ke field di bawah
3. Klik "Sinkronisasi" untuk mulai

🔗 Konfigurasi Google Apps Script

Contoh: https://script.google.com/macros/d/1234567890ABCDEFG/usercontent

Contoh ID dari: https://docs.google.com/spreadsheets/d/1234567890/edit

📋 Panduan Setup Google Apps Script

STEP 1: Buat Google Sheet Baru

  1. Buka Google Sheets
  2. Klik "Buat spreadsheet baru"
  3. Beri nama: "Audit & Evaluasi Dosen"
  4. Copy ID dari URL (bagian setelah /d/)

STEP 2: Buat Google Apps Script

  1. Di Sheet yang baru, buka Tools → Script editor
  2. Copy-paste kode di bawah ke Apps Script editor
  3. Klik Save (Ctrl+S)
  4. Klik "Deploy" → "New deployment"
  5. Pilih type: "Web app"
  6. Execute as: Pilih akun Google Anda
  7. Who has access: "Anyone"
  8. Klik Deploy → Copy URL yang muncul

📋 CARA COPY KODE (PENTING!):

  1. Klik tombol "📋 Copy Kode" di bawah
  2. Klik di dalam textarea kode (area putih)
  3. Tekan Ctrl+A untuk select semua
  4. Tekan Ctrl+C untuk copy
  5. Buka Google Apps Script editor di tab lain
  6. Delete semua kode yang ada (Ctrl+A → Delete)
  7. Paste kode baru (Ctrl+V)
  8. Tekan Ctrl+S untuk save

❓ Troubleshooting Jika Ada Error

Error: "Authorization required"

  • → Di Apps Script editor, klik tombol "Run" untuk authorize
  • → Pilih akun Google Anda
  • → Klik "Allow"

Error: "Exceeds size limit"

  • → Data terlalu besar untuk satu kali upload
  • → Bagi data menjadi beberapa batch (max ~500 dosen per upload)

Error: "Cannot find method clearContents"

  • → Ini adalah error versi Apps Script
  • → Ganti "clearContents()" dengan "clear()"

Koneksi timeout / blank response

  • → Refresh page dan coba lagi
  • → Cek apakah Apps Script masih aktif (jangan dihapus)
  • → Deploy hanya sekali (jangan multiple deploy)
